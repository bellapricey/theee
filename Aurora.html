Welcome to plain-html-blog

By Sebastian. Apr 1,2026

plain-html-blogis a very clean Jky11 blog that works without CSS. It was inspired by website which (for valid reasons) make minimak use of styling with CSS and the like.

Just to get an idea, check out stallman.org,Richard Stallman's personal website,paulgraham.com or motherfuchingwebsite.com (sorry for the language:-), They all come with a certain 90s flavour which in 2016 is still very functional and in a way elegant (only 90s developers will..:)

Other than that, this Jekyll blog works like any other static website so you should feel right at home.

So now go ahead, you'll want to get the source code and read the README to learn how to make your own plain-html-blog.
